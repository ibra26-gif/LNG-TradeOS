HOW TO READ THIS SECTION
ROLLING TAB
Bar height = Pearson r or Spearman r. Range -1 to +1. Solid bars = statistically significant (p<0.05). Faded bars = insufficient data to trust the number. A bar near 0 = indices moved independently. Near +1 = moved together. Negative = moved in opposite directions.

Stability sigma: std dev of rolling r over time. Low (e.g. 0.05) = consistent relationship. High (e.g. 0.25) = correlation regime switches between periods.
ANALYSIS TAB — LAG / LEAD
X-axis = lag in days (-10 to +10). Peak bar = optimal lag. If peak is at +2, Index A today predicts Index B in 2 days. If peak is at 0, the indices move synchronously with no timing edge.

A lag gain of <0.05 over synchronous r is noise. A gain of >0.10 is worth investigating as a timing signal.
ANALYSIS TAB — SCATTER
Each dot = one day. Orange line = regression. r = relationship strength. R2 = variance explained: R2=0.55 means one index explains 55% of the other's daily moves. Beta slope = how much B moves per unit of A. Tight cluster = reliable. Dispersed cloud = weak or regime-dependent regardless of r.
MATRIX TAB
Cell: r (top) + sigma stability (bottom). Blue intensity = positive correlation. Red = negative. Faded = not significant, treat as zero. Cross-tenor: compare different delivery months — high r across tenors means the arb relationship persists regardless of delivery timing.

DETECT scans for: breakdowns (r dropped >0.25 vs 90D avg), developing correlations, unusual lags, and insignificant cells.
